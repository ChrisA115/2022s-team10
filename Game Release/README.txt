RISE OF THE DAMNED
------------------
A vertical roguelike platformer 
where you battle you way out of hell.

​Controls:
WASD/Space​ - Movement
R - Pick up/swap item
Arrow Keys - Attack
F - Switch attacks

-Installation Instructions-
Unzip all of the files, then run
the executable ("Rise of the Damned.exe").
